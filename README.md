# 🚀 JavaScript DSA Challenge

Welcome to my JavaScript DSA (Data Structures and Algorithms) Challenge!  
This repository is a collection of DSA problems solved using JavaScript, with a strong focus on writing clean, understandable code. Each file contains code examples along with **time complexity** and **space complexity** as comments for better understanding and learning.

---

## 📂 Repository Structure

Each `.js` file in this repo represents a different DSA problem or concept.


Each file includes:
- ✅ The problem statement (as a comment)
- ⏱️ Time Complexity (Big O notation)
- 💾 Space Complexity
- 🧠 Clear, readable solution using core JavaScript

---

## 🧠 Why This Challenge?

- To sharpen my problem-solving skills
- To build a strong foundation in JavaScript-based DSA
- To prepare for coding interviews and competitive programming
- To help others learn through well-documented code
